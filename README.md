# Online Quiz Maker

A web-based application for creating and taking quizzes, built with MERN stack (MongoDB, Express.js, React, Node.js).

## Features

- User registration and login with role-based access (Admin, Teacher, Student)
- Quiz creation with multiple questions
- Timed quizzes with automatic submission
- Difficulty levels and scoring
- Leaderboard and ranking system
- Certificate generation
- Analytics dashboard
- Anti-cheating mechanisms
- Multi-language support
- PWA support
- And many more advanced features...

## Project Structure

- `frontend/`: React application
- `backend/`: Node.js/Express server
- `database/`: Database schema and scripts
- `docs/`: Diagrams and documentation

## Setup

1. Clone the repository
2. Install dependencies for frontend and backend
3. Set up MongoDB database
4. Run the application

## Technologies

- Frontend: React, Bootstrap
- Backend: Node.js, Express.js
- Database: MongoDB
- Authentication: JWT
